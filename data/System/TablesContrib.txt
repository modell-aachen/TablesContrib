---+!! TablesContrib

%SHORTDESCRIPTION%

Foswiki 1.2.0 introduces a new internal API that supports the parsing and modelling of in-topic tables. This is designed to canonicalise the handling of tables in the core and extensions. This module makes the tables code available to Foswiki versions prior to 1.2.
